# Dummy Text Generator 

A lightweight, browser-based Lorem Ipsum generator.
 
🌐 **[Live demo](https://gsandrini.github.io/dummy-text/)**

---
 
## Features
 
- Generate text by **words**, **paragraphs**, **characters** or **bytes**
- Quick presets and adjustable quantity
- One-click **copy to clipboard**
- 100% client-side - runs entirely in the browser
- Dark mode interface
- Zero runtime dependencies beyond Alpine.js
 
---

## Usage
 
### As a web app
 
Open the [live demo](https://gsandrini.github.io/dummy-text/) in any modern browser.

---
 
### Install
 
The app is available as a **Progressive Web App (PWA)** and can be installed directly on your device.
 
1. Open the [live demo](https://gsandrini.github.io/dummy-text/) in your browser
2. Look for the **install icon** (⊕) in the address bar, or open the browser menu and select **"Install app"** / **"Add to Home Screen"**
3. Confirm the installation - the app will appear on your desktop or home screen and work just like a native app, even offline
